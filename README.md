# CoffeeBot
A coffee cooking robot using an Universal Robot.

This project uses a Universal Robot to brew coffee using a V60. The brewing technique is inspired by the 1 cup V60 brewing recepite by James Hoffman (https://www.youtube.com/watch?v=1oB1oDrDkHM).

This Github repository explains in 3 steps how the (1) process engine is set up, (2) mqtt service is set up and connected and (3) harware is design. All relevant files including the CAD files, the code for the robot and the process are provided. Aditionally, a video and pictures of the process and the parts are provided.
